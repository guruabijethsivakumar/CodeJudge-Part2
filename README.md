# CodeJudge Part 2 - SQL Query Implementation

This repository contains Part 2 solution for the CodeJudge DBMS assignment.

Files included:

- queries.sql -> all SQL queries
- query_outputs.md -> outputs, validation notes, and summaries
- sql_reasoning.md -> SQL explanation and reasoning
- data/ -> raw dataset
- codejudge_raw.db -> SQLite database

Tools used:
- Python
- SQLite
- VS Code
- GitHub

The queries were tested on the imported dataset and validated using actual outputs.
